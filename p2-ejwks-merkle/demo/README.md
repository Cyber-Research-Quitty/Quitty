# Demo Folder

This folder is for presentation and demo material related to the P2 dashboard.

Runtime dashboard assets live in `app/demo/`.
This top-level `demo/` folder is for:
- presentation notes
- sample dashboard payloads
- screenshots or screen recordings

Contents:
- `presentation_notes.md`: short explanation of what to show during a demo
- `sample_dashboard_snapshot.json`: trimmed sample from `/dashboard/data`
- `screenshots/.gitkeep`: placeholder for captured images

Useful URLs during a live demo:
- `http://127.0.0.1:8200/dashboard`
- `http://127.0.0.1:8200/dashboard/data`
